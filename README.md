# Bayanihan App

A gamified community volunteering platform designed to encourage people to help their communities through task-based participation, rewards, and friendly competition.

## 📌 About the Project

**Bayanihan App** is a community volunteer website where users can create volunteer tasks and invite others to participate. The platform transforms volunteering into an engaging experience by adding gamification elements such as points, rewards, and leaderboards.

Users can contribute to their communities, track their impact, and earn recognition for their efforts.

## ✨ Features

* Create and post volunteer tasks
* Join and participate in community activities
* Earn points by completing volunteer work
* Redeem points for rewards
* Community contribution tracking
* User leaderboard system
* Gamified volunteering experience

## 🎮 How It Works

1. Users create volunteer tasks for their community
2. Other users browse available tasks and participate
3. Completed volunteer activities reward users with points
4. Points can be exchanged for available rewards
5. Top contributors appear on the leaderboard

## 🏆 Gamification System

The app encourages participation through:

* **Points System**
  Earn points by helping with volunteer activities

* **Rewards System**
  Exchange earned points for available rewards

* **Leaderboard**
  Compete with other volunteers and recognize top contributors

## 👥 User Roles

### Volunteer Users

* View available tasks
* Join volunteer activities
* Complete tasks
* Earn points
* Redeem rewards

### Community Organizers

* Create volunteer tasks
* Manage activities
* Track participants
